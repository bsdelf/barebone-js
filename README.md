# barebone-js

barebone-js is a backend boilerplate for Node.js and TypeScript.

## Features

- TypeScript
- Graceful kill
- Stacked configurations
- Extensible application context
- Well-organized project structure
- Almost zero learning curve

## Project Structure

```
└── src
    ├── applications
    ├── commands
    ├── context
    ├── controllers
    ├── libraries
    ├── migrations
    ├── models
    ├── providers
    └── routes
```
